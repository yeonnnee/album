# Album
Upload Image Application with React

<br />

### Period
2022-07-14 ~ 2022-07-16


### Stacks 
- React
- Redux-toolkit
- RTK query
- scss
- typescript



### Features

- each post with a title and an image
- 5 posts per page
- pagination
- search by album title
- CRUD

### API 
- posts: https://jsonplaceholder.typicode.com/albums
- image: https://place-hold.it/

### Get Started

```
npm i
npm run start
```
